## 🚀 部署到 Render（免费）

### 第一步：上传到 GitHub

```bash
# 在 E:\金丝猴网页 目录下执行
git init
git add auth_server.py index_cloud.html requirements.txt Procfile render.yaml .gitignore
git commit -m "Initial commit"
```

然后创建 GitHub 仓库并推送：
```bash
git remote add origin https://github.com/你的用户名/golden-monkey.git
git push -u origin main
```

### 第二步：在 Render 部署

1. 访问 https://render.com
2. 用 GitHub 账号登录
3. 点击 "New +" → "Web Service"
4. 选择你的 `golden-monkey` 仓库
5. 配置会自动从 `render.yaml` 读取
6. 点击 "Create Web Service"

### 第三步：设置环境变量

在 Render 后台添加以下环境变量：

```
SMTP_HOST = smtp-mail.outlook.com
SMTP_PORT = 587
SMTP_USER = 你的邮箱
SMTP_PASSWORD = 你的授权码
SMTP_FROM_NAME = 金丝猴智能平台
```

### 第四步：获取你的云端地址

部署成功后，Render 会给你一个地址：
```
https://golden-monkey-api-xxxx.onrender.com
```

### 第五步：更新前端

打开 `index_cloud.html`，修改第 113 行：
```javascript
const API_BASE = 'https://golden-monkey-api-xxxx.onrender.com';  // 改成你的地址
```

完成！现在任何人都可以访问了！

2. 首次访问需要 30 秒唤醒
3. 数据库存在内存中，重启会丢失（可改为 SQLite 文件）
